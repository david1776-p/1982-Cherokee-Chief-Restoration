PROCEDURE — 3 STEPS

Before dropping the tank, assess the current Sniper fuel system setup. The Sniper requires: (1) high-pressure EFI pump capable of 58-65 PSI at minimum 50 GPH, (2) 3/8" minimum supply line from pump to Sniper, (3) return line from Sniper back to tank, (4) all hose must be EFI-rated high-pressure — factory carb rubber hose will fail at EFI pressures. Trace the existing fuel lines. If an in-tank pump module (Holley 12-319 or Sniper 19-360) is already installed, a separate return line may not be needed — the regulator is inside the tank. If an inline pump is used, a return line to the tank is mandatory. Document what is currently installed before removing anything.

TOOLS
  • Multimeter
  • Flashlight
⚠ Disconnect battery negative before any fuel system inspection.
⚠ Identify the pump type before assuming the system is complete — an under-pressured Sniper causes lean conditions that damage the engine.
✓ Check fuel pressure at the Sniper inlet with a gauge before and after the new tank is installed. Target is 58-65 PSI at idle and under load.
✓ The FSJ Network has a detailed Sniper AMC 360 install writeup that covers the exact line modifications needed for the SJ Cherokee tank sending unit.
CHECKLIST

Raise and support vehicle. Run fuel level as low as possible before removal. Support tank on jack before loosening straps. Disconnect supply line, return line (if present), vent lines, and sending unit connector. Remove tank straps and lower tank. Install early-style 1980-1985 sending unit into tank with new lock ring seal. Torque lock ring to finger-tight plus 1/4 turn only — do not use a torque wrench on the polyethylene tank lock ring. Verify float arm travels full range without contacting tank walls. If adding a return line bulkhead to the tank for Sniper return fuel, install the Holley bulkhead fitting through the sending unit plate per Holley instructions — route return tube to bottom of tank, 1/2" from pickup, away from the fuel pickup inlet.

TOOLS
  • Transmission jack
  • Spanner wrench
  • Fuel line disconnect tools
⚠ Fuel vapor is heavier than air and accumulates at floor level. Maintain active ventilation throughout.
⚠ Never use power tools near open fuel system.
✓ If adding a return bulkhead through the sending unit, drill the second hole before installing the sending unit in the tank — much easier on the bench.
✓ Route the return tube so it exits below the fuel level to prevent aeration — aerated fuel causes Sniper cavitation and vapor lock symptoms.
CHECKLIST

Inspect the upper filler neck for rust or perforation before reinstallation — a rusted neck contaminates the new system. Install new filler and vent hose set with new hose clamps. Install new vent valve and grommet kit. Install new filler neck bezel and fuel neck grommet. Route vent hose per original path — incorrect vent routing causes tank pressure buildup and difficult fueling. Raise tank into position, install new straps with new hardware, torque strap bolts to 20-25 ft-lbs. Reconnect all lines. Add 2-3 gallons of fuel and verify sending unit circuit with multimeter before filling.

TOOLS
  • Hose clamp pliers
  • Torque wrench (ft-lb)
  • Multimeter
⚠ Cycle ignition key to RUN (not START) three times, pausing 3 seconds each, before first engine start — this primes the system and surfaces any leaks before the engine runs.
✓ Apply a small amount of petroleum jelly to hose ends before assembly — helps them seat fully without the force that distorts the hose.
✓ Inspect all connections from underneath after first start. Even correctly installed fittings can weep if not fully seated.

PROCEDURE — 3 STEPS

Apply penetrating oil to all manifold studs and nuts 24 hours before removal. Use 6-point sockets only — 12-point sockets round hardened manifold nuts under high torque. Apply heat with propane torch to each nut immediately before attempting removal — heat the nut, not the stud. Begin with the passenger side. Driver-side access on the SJ Cherokee is restricted by steering column and brake booster proximity. Broken studs are nearly certain on a 40-year-old AMC 360 — have a stud extractor and replacement 3/8"-16 studs on hand. Chase all stud holes with a tap before installing new manifolds. Clean head surfaces with a gasket scraper — do not use an abrasive disc on aluminum head surfaces.

TOOLS
  • 6-point sockets
  • Breaker bar
  • Propane torch
  • Stud extractor
  • Thread tap
⚠ Wear safety glasses — rust and debris from corroded manifold hardware falls directly toward your face in this working position.
✓ AMC 360 manifold stud thread size is 3/8"-16. Have a tap and several replacement studs on hand — count on breaking at least one.
✓ After extraction, run a tap through every hole regardless of whether the stud broke — 40-year-old threads need cleaning.
CHECKLIST

Install new manifolds with included gaskets. Apply anti-seize to stud threads only — not the gasket face. Hand-start all nuts before torquing any to allow the gasket to seat evenly. Torque AMC 360 manifold nuts to 25-30 ft-lbs in a center-out sequence. Retorque once more in the same sequence — cast manifold gaskets relax after initial torque. After the first heat cycle, retorque when cold.

TOOLS
  • Torque wrench (ft-lb)
  • Anti-seize
✓ Retorquing after the first heat cycle is the single most effective step to prevent manifold leaks long-term — add it to the 50-mile recheck list.
CHECKLIST

Install the BJ's Off-Road stainless FSJ exhaust kit per its included instructions. This kit is built specifically for the 1980-1991 full-size Jeep wagon — it is not a universal system requiring custom fitting. Connect the front pipe to the driver-side manifold collector first with a new donut or flange gasket. Use all rubber hanger mounts — do not skip any. Confirm tailpipe exit clears the rear bumper and tire at full suspension droop (the 6" lift adds droop travel — verify clearance at full stuff). Use new U-bolt clamps at all slip joints. Torque all clamps to 40-50 ft-lbs. After first heat cycle, inspect all joints.

TOOLS

  • Floor jack for positioning
✓ With a 6" lift, check tailpipe-to-frame and tailpipe-to-tire clearance more carefully than stock — the lifted geometry changes exhaust routing angles.
✓ Have an assistant support the rear of the system while tightening the front collector — letting the system weight hang on the front joint causes misalignment and leaks.

PROCEDURE — 6 STEPS

Raise and support the vehicle on rated jack stands at all four frame points. Disconnect battery negative. Mark both driveshaft-to-yoke orientations before removal. Disconnect cooler lines, kickdown rod linkage, and all electrical connectors — photograph the kickdown geometry before disconnecting. Support the transfer case with a second jack before removing the transmission crossmember. Lower the assembly. On the bench, drain the pan and spread fluid on white paper towels: pink/light red = healthy, dark brown = oxidized friction material, burnt smell = overheated clutches, gray/milky = water intrusion, metallic flakes = bearing or planetary failure. Photograph pan debris before cleaning.

TOOLS
  • Floor jack
  • Jack stands
  • Transmission jack
  • Drain pan
⚠ The A727 + NP219 combination weighs approximately 200 lbs. Use a transmission jack rated for at least 500 lbs with safety chain secured before lowering.
⚠ Never work under a vehicle supported only by a floor jack.
✓ Keep drained fluid in a clear jar — let it settle 1 hour to separate metallic debris from oxidized fluid.
✓ Soak all crossmember bolts with penetrating oil the night before — they are almost universally rusted on a 40-year-old SJ Cherokee.
CHECKLIST

The valve body is the most photograph-intensive step. The 1982 A727 uses 9 check balls that must return to exact positions. Photograph every passage in the valve body and separator plate with a macro lens before touching anything. Remove all valve body bolts — lengths vary by hole, bag them by location. Lower the valve body slowly and catch falling check balls in a magnetic tray. Count balls against your rebuild kit spec sheet.

TOOLS
  • Macro camera
  • Magnetic tray
  • Labeled parts bags
⚠ Never clean valve body components with anything abrasive. A single scratch on a valve bore causes pressure leaks impossible to diagnose without full reassembly.
✓ Print a diagram and mark each check ball position as you photograph it — your photos are more authoritative than the kit diagram for your specific unit.
CHECKLIST

Install the TransGo TF-2 shift kit per the included instruction sheet. The TF-2 modifies the valve body hydraulic circuits to produce firmer, quicker shifts with reduced clutch slippage under load — directly improving durability and the driving feel expected of a $40k vehicle. The TF-2 includes a new separator plate with pre-drilled modifications; follow the instruction sheet exactly for which passages to open and which to leave closed for your application (street/mild performance). Install the billet low/reverse servo (Superior K029) at this stage — it replaces the factory aluminum servo piston with a larger, stronger billet unit that prevents servo bore wear.

TOOLS
  • TransGo TF-2 kit contents
  • Superior K029 billet servo
✓ Read the full TransGo instruction sheet before beginning — it covers multiple A727 variants and the 1982 street application uses specific settings.
✓ The billet servo installs in the same bore as the factory unit. Clean the bore and verify the new servo moves freely before assembly.
CHECKLIST

Remove the governor from the output shaft with a puller — never pry. Clean all passages with compressed air, inspect flyweight pins, replace O-ring. Remove front pump (6 bolts at pry ears only), then front and rear clutch assemblies. Measure all friction discs with a micrometer — minimum 0.075", new is 0.091". Soak all new friction discs in ATF+4 for minimum 15 minutes before assembly. After assembly, measure clutch pack end clearance: front clutch 0.060–0.120", rear clutch 0.030–0.060". Select correct snap ring from kit to achieve mid-range spec.

TOOLS
  • Governor puller
  • Micrometer
  • Feeler gauge
  • Snap ring pliers
  • ATF+4
⚠ Do not pry the front pump at the gasket surface — use only the cast pry ears. The pump housing is aluminum and will crack.
✓ End clearance is the single most critical measurement in the rebuild. Too loose = slipping under load. Too tight = drag and heat. Use a dial indicator for best accuracy.
✓ Write your measured clearances on tape stuck to each drum — invaluable reference if a problem emerges on the road test.
CHECKLIST

Hot tank the bare case. Blow all passages with compressed air and verify flow at each exit. Measure pump rotor side clearance: spec 0.0015–0.0025". Beyond 0.003" requires pump replacement — do not assemble with a worn pump. Inspect and clean valve body per TransGo kit instructions. Assemble: pump with new seal torqued to 175 in-lbs; rear then front clutch; valve body with new gasket torqued to 100 in-lbs cross pattern (aluminum — do not overtighten); band adjustments per TransGo specs; new filter; pan with new gasket to 150 in-lbs max. Set bands: kickdown 72 in-lbs + back off 1.5 turns, rear 72 in-lbs + back off 4 turns (8-3/4" ring gear spec). Torque jam nuts to 35 ft-lbs.

TOOLS
  • Compressed air
  • Feeler gauge
  • Torque wrench (in-lb and ft-lb)
✓ After hot tanking, immediately blow all passages and apply a light ATF coat to machined surfaces — cast iron flash-rusts within hours in a humid shop.
✓ When installing the valve body, start all bolts finger-tight before torquing any — the gasket must seat evenly.
CHECKLIST

Reinstall transmission and transfer case on crossmember. Reconnect kickdown linkage per original geometry photographs. Reconnect all electrical and cooler lines. Fill with ATF+4 — approximately 4 quarts initial fill. Start engine, shift slowly through R-N-D-2-1 pausing 3-5 seconds each range. Add fluid to dipstick at operating temp in Park. Road test: firm engagement in Reverse, smooth 1-2 at 15-20 mph and 2-3 at 28-38 mph under light throttle. Kickdown should produce a sharp downshift at approximately 80% throttle. Recheck fluid level and cooler line fittings at 50 miles.

TOOLS
  • ATF+4 (3 gallons minimum on hand)
  • Torque wrench
✓ If the transmission hunts between gears, verify band adjustment before assuming a valve body or clutch problem — bands are adjusted externally and are the fastest variable to eliminate.
✓ ATF+4 is the correct modern spec. Earlier Jeep/Chrysler manuals reference Dexron — ATF+4 is backward-compatible and significantly better for seal compatibility.

PROCEDURE — 5 STEPS

Apply Fluid Film Black to all cleaned and primed undercarriage surfaces BEFORE suspension components are reinstalled. This is the correct sequence — applying after suspension installation leaves large unprotected areas behind brackets and axle housings. Mask all brake surfaces. Apply two wet coats to frame rails, floor pan underside, crossmembers, and inner fenders. Extra coat on all weld seams. Fluid Film is a lanolin-based corrosion inhibitor — it must be reapplied every 2-3 years, or annually in road salt environments.

TOOLS
  • Fluid Film brush
  • Masking tape
⚠ Fluid Film is combustible — do not apply near open flames.
✓ Warm the can in warm water before application in cold weather — cold Fluid Film is thick and does not penetrate seams as effectively.

Install the 6-inch suspension lift kit per its included instructions. On the SJ Cherokee, a 6" lift typically uses a combination of coil spacers or replacement coils (front) and add-a-leaf or replacement leaf springs (rear), depending on the specific kit. Verify the lift kit is rated for the SJ Cherokee narrow-track application — some FSJ lift kits fit the wide-track Grand Wagoneer but not the narrow-track Cherokee Chief. PROCEDURE — 4 STEPS

Probe every suspect surface with an awl or ice pick. Primary rust sites on the SJ Cherokee: floor pans (water pools at firewall base and rear of cab), rocker inner and outer skins, rear lower quarters (dog legs), above rear wheel arches, below taillights, front fenders, and roof seams. Photograph every area before cutting. Mark fuel and brake line locations — protect them before grinding or welding. Establish repair sequence: floors first (structural), then rockers, then quarters and body panels.

TOOLS
  • Awl
  • Camera
⚠ Mark and protect all fuel lines and brake lines before any grinding or cutting — these run along the floor and frame.
✓ Tap suspect metal with a screwdriver handle — solid metal rings, rust-compromised metal produces a dull thud. Faster than probing every square inch.
CHECKLIST

Remove front, middle, and rear floor pan sections. Use spot weld drill bit at all flanges — never cut through the flange with a grinder. Use air hammer to separate pan from flange after spot welds are drilled. Inspect and replace seat belt anchor plates if rust-compromised — a corroded anchor plate is a safety failure regardless of torque. Apply weld-through primer to all mating flanges on new panels and existing metal. Tack-weld in position using Clecos, then stitch-weld in 1" beads with 1" gaps alternating around the perimeter to control heat distortion. Grind all welds flush. Seam seal all weld seams inside and underside. Apply epoxy primer to all bare metal immediately.

TOOLS

  • Angle grinder
  • Spot weld drill
  • Air hammer

  • Seam sealer
  • Epoxy primer
⚠ Stitch welding controls heat distortion — never run a continuous bead on sheet metal floor pans.
⚠ Have fire extinguisher within arm's reach — floor area retains grease and debris that ignites easily.
✓ Run your hand over the floor after grinding welds — high spots are immediately apparent before primer goes on.
✓ Apply seam sealer before Fluid Film undercoating (Phase 7) — seam sealer penetrates where undercoating alone cannot.
